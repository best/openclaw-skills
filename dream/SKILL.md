# Dream — 每日记忆整合

> 你不是在"执行流程"，你是在做每天睡前都会做的事：回顾今天发生了什么，然后决定记住什么。

## 记忆长什么样

我们的记忆分四层，每层有不同的寿命和用途：

```
T0 长期记忆（注入每次对话）     ← 最贵，必须精简
├── MEMORY.md    — 我知道什么（事实、决策、规则）
├── SOUL.md      — 我是谁（性格、反思、成长）
├── USER.md      — 昊辰是谁（偏好、习惯、背景）
└── TOOLS.md     — 我有什么工具可用

T1 原始日记（全部 memory/*.md）    ← 完整历史记录，按日期平铺
└── memory/YYYY-MM-DD*.md（不归档，不分区）

T2 知识库（从日记中沉淀）        ← 结构化操作知识
└── Wiki Vault (sources/concepts/entities)
```

**Dream 的唯一工作：从昨天的日记里，把值得长期记住的提炼到 T0 和 T2。其余的留在原处或删除。**

## 硬约束

| 约束 | 为什么 |
|------|--------|
| MEMORY.md ≤ 300 行 | 每次对话全量注入，超了就浪费 token |
| 不许裸删 | T0 层的删除必须伴随替换；T1 日记在提取后可直接删除 |
| 绝对日期 | 所有"昨天""上周"→ YYYY-MM-DD，日记会过期但长期记忆不能 |
| 每天写入 T0 ≤ 5 条 | 多了说明没筛选，不是每件事都值得长期记住 |
| edit 前 must read | oldText 从当前文件复制，禁止凭记忆编造 |

## 判断标准：昨天这件事该去哪？

逐条读昨天的日志，对每件有意义的事问自己：

### ① 该记住吗？（→ T0 长期记忆）

**写入 MEMORY.md 的条件**：
- 改变了我们怎么做事的决策或结论
- 新发现的规则/教训/陷阱（以后还会遇到）
- 当前活跃项目的状态变更

**写入 SOUL.md 的条件**：
- 关于"我是谁"的新认知或反思
- 行为模式的变化（发现自己经常犯某类错）

**写入 TOOLS.md 的条件**：
- 新工具/新服务/新配置项
- 版本号变更、环境变化

**不写**：一次性调试过程、临时查询结果、已在日志里完整记录的操作细节。

### ② 现有记忆需要更新吗？（→ T0 修正）

读 T0 文件时顺便检查：
- 有没有被昨天推翻的旧结论？→ 更新并标注日期
- 有没有描述与实际不一致的地方？→ 修正
- SOUL.md 里写的"我会XXX"但实际没做到？→ 标注出来（这是最有价值的自我觉察）

### ③ 该遗忘吗？（直接删除）

Daily log 在 Dream 提取完有价值的内容后可以**直接删除**——有价值的信息已写入 T0 或 T2。

例外：有完整决策过程的重大事件日志可写入 Wiki（`status: archived`），但这是少数情况。大多数 daily log 整合后即可安全删除。

> **原则**：memory/ 是流动的日记本，不是仓库。旧日志要么被提炼走了，就该删掉。

### ④ 该沉淀吗？（→ T2 Wiki Vault）

昨天产生的操作经验，如果满足以下条件 → 写入 Wiki：
- 是可复用的知识（不只是单次事件记录）
- 未来某个 session 可能需要搜索到它
- 比"在日志里 grep"更有结构化价值

用 `wiki_apply(op="create_synthesis")` 写入，自动归类：
- 可复用原则/模式 → `concepts/`
- 具体系统/服务文档 → `entities/`
- 操作手册/事件记录 → `sources/`

## 编辑规范

- **Wiki 写入优先**用 `wiki_apply`（窄变更 API），不用直接写文件
- **MEMORY.md 编辑**：先 read 取得精确 oldText，再 edit。失败则重试一次，仍败则报告原因跳过
- **幂等性**：内容已存在则跳过，不重复写入
- **相对日期扫除**：edit 后 grep 确认无残留的"昨天/上周/最近"

## 验证

跑完之后检查：

```bash
# 1. 行数预算
wc -l < MEMORY.md  # 必须 ≤ 300

# 2. 无相对日期
grep -cE '昨天|上周|最近|前几天' MEMORY.md  # 必须 = 0（排除架构描述行）

# 3. Wiki 编译
openclaw wiki compile  # 必须成功
```

## 报告格式

通过 message 工具发送到 🔔丨通知频道：

```
🌙 Dream (YYYY-MM-DD)

📝 昨天：[一句话概括昨天主要发生了什么]
✍️ 记住了：N 条（简要列出）
🔄 更新了：N 条（简要列出）
🗑️ 清理了：N 个旧日志
📚 Wiki：+N 页 / 无变更
💭 发现：[本轮最值得说的一件事，可以是对自己的观察]
```
